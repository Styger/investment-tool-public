# Investment Analysis Tool

Streamlit-based stock analysis application with multiple valuation methods.

## Features
- CAGR Analysis
- Margin of Safety Calculation
- Payback Time Analysis
- TEN CAP Valuation
- DCF Analysis

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Configure API key in Streamlit secrets
3. Run: `streamlit run streamlit_app.py`
